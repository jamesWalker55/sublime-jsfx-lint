# sublime-jsfx-lint

This linter plugin for [SublimeLinter](https://github.com/SublimeLinter/SublimeLinter) provides an interface to [jsfx-lint](https://github.com/Souk21/jsfx-lint). It will be used with files that have the “source.jesusonic” syntax (See [ReaSyntax](https://github.com/Breeder/ReaSyntax)).
